# Project 02: Multi-Modal "Vision" Assistant

## Overview
An intelligent assistant capable of understanding and reasoning over **Images and Videos**. It can analyze visual data alongside technical documentation to provide high-accuracy guidance.

## Key Features
- **Visual Reasoning**: Answer questions about what is happening in an image or video.
- **Cross-Modal Retrieval**: Match visual patterns with text descriptions in a manual.
- **Real-time Analysis**: Processes live feeds or uploaded media.

## Tech Stack
- **Vision Model**: Gemini 1.5 Pro / GPT-4o
- **Video Processing**: OpenCV / MoviePy
- **UI**: Streamlit with media playback
- **Framework**: LangChain Multi-modal

## Use Cases
- **Industrial Repair**: Showing a camera to a broken part to get repair steps.
- **Accessibility**: Real-time scene description for the visually impaired.
- **Security**: Summarizing events from a CCTV clip.
